Editable worksheet for the remaining program products that need a
package size (or a standalone price) before a cost basis can be derived.
Fill in `inputValue` for each row, then run the calc script to preview
the cost-per-unit.

## How to fill it in

1. Open the JSON file. Each entry has `inputValue: null`.
2. Set `inputValue` to the package size, in the unit shown by `inputUnit`:
   - `gal/case` → gallons per case
   - `lb/bag`   → pounds per bag
   - `lb/pack`  → pounds per pack
3. Save, then run:
   ```
   node scripts/calc-crosswinds-package-size-inputs.mjs
   ```
   It prints a preview cost-per-unit per filled row. **Nothing is
   applied** — `applyEligible` stays false.
4. For bottle / bundled products, get a **standalone price** (and bottle
   size) first — a package size alone is not enough.

- **Harmony** — 5 cases @ $842.10, set `inputValue = 2.5` (gal/case):
  `842.10 / (5 × 2.5) = $67.37 / gal`

  `1144.00 / (40 × 50) = $0.57 / lb`

  `375.00 / (10 × 2) = $18.75 / lb`
